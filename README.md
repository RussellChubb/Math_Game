## ğŸ¯ How to Play

1. **Start Game**: Launch from the main menu
2. **Countdown**: A 3-second countdown prepares you for action
3. **Solve Problems**: Type your answer and press Enter
4. **Scoring**:
   - Correct answer: +1 point
   - Incorrect answer: -1 point
5. **Time Limit**: Complete as many problems as possible in 60 seconds
6. **Game Over**: Enter your name (*up to 4 characters*) to save your score
